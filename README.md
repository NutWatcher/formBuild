#formbuilder

##安装依赖插件

请先保证安装了Node4.0+ 、 Npm3.0+

安装依赖命令

```npm install```

##使用方法

###介绍

项目使用了React+Redux的框架，使用webpack进行打包

###开发模式

命令
```npm start```

浏览器打开 `http://localhost:9093/public/index.html`



###打包模式
命令
```npm run bulid```

打包js路径在 `/dist` 下。

`/public/*.html` 中可自行替换开源框架cdn
